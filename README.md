# 🎞 movies
일일 박스오피스 순위를 제공하고 찾고싶은 영화나 컨텐츠를 카테고리별로 검색하여 영화 상세정보를 확인할 수 있는 영화 검색 웹 서비스입니다.

### 결과물

### 소개
- 일일 박스오피스 순위 제공
- 제목, 감독, 키워드별로 원하는 영화 검색
- 장르별 영화 검색
- 영화에 대한 상세한 정보(관람등급, 개봉일, 감독, 출연진 등) 제공
- 비슷한 영화 추천

### 사용 기술
- vue.js
- vuex
- vue-router
- Axios
- javascript
- html
- css
- Netlify

### 프로젝트 정보
- vue-cli version
	```bash
	@ vue/cli 4.3.1
	```
- 실행방법
	```bash
	- node module install  
	$ npm i or npm install

	- serve
	$ npm run serve

	- build
	$ npm run build
	```
